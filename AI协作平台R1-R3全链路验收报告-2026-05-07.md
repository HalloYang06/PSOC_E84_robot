## 三级审核生效铁证

`broadcast/preview` 返回的 `review_decisions` 字段：

```json
[
  { "requires_review": false, "source": "workstation", "seat_id": "前端工位", "policy": "skip" },
  { "requires_review": true,  "source": "npc",         "seat_id": "执行工位", "policy": "force" }
]
```

| 优先级 | 工位级 skip | NPC 级 force | 实际 |
|---|---|---|---|
| 前端工位 → NPC1 (inherit) | 工位说 skip | NPC 说 inherit | **免审，source=workstation** ✓ |
| 执行工位 → NPC2 (force) | 工位继承项目 | NPC 说 force | **强审，source=npc**（NPC 优先级最高）✓ |

→ NPC > 工位 > 项目 default 的优先级链路打通。
