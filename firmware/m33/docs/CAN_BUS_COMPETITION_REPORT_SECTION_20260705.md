# CAN 总线通信质量量化指标

## 1. 测试目的

为验证康复机械臂控制系统中 CAN 总线通信的实时性、稳定性和可靠性，本项目对 M33 主控、F103 传感节点、NanoPi 心跳节点以及多路电机反馈进行了实测量化。测试重点不是只判断“能不能通信”，而是用可复现的指标评估总线负载、帧周期稳定性、端到端响应延迟、应用层丢包、控制器错误计数和队列溢出情况。

## 2. 测试条件

| 项目 | 配置 |
|---|---|
| 主控 | Infineon PSOC Edge E84 / RT-Thread M33 |
| CAN 类型 | Classic CAN |
| CAN 波特率 | 1 Mbps |
| 采样方式 | M33 固件内置 CAN metrics 统计模块 |
| 采样窗口 | 30 s 稳态采样；请求响应延迟单独采样 |
| 统计对象 | F103 传感帧、NanoPi 心跳、M33 状态帧、4/5/6 号电机主动反馈 |
| 安全边界 | 电机侧仅打开主动反馈，不发送使能、位置、速度、力矩或置零命令 |

## 3. 核心量化结果

| 指标 | 实测结果 | 评价 |
|---|---:|---|
| 基础传感通信总线利用率 | 约 1.3% | 负载很低，带宽余量充足 |
| 接入 4/5/6 号电机主动反馈后总线利用率 | 约 5.7% | 多电机反馈下仍有较大余量 |
| 基础传感通信帧率 | 100.79 frames/s | 稳定 |
| 接入 4/5/6 号电机主动反馈后帧率 | 381.19 frames/s | 稳定 |
| F103 传感帧周期 | P50/P95/P99/max = 20/20/20/20 ms | 50 Hz 稳定传输 |
| 4 号电机反馈周期 | P50/P95/P99/max = 10/10/11/11 ms | 约 100 Hz 稳定反馈 |
| 5 号电机反馈周期 | P50/P95/P99/max = 10/10/11/11 ms | 约 100 Hz 稳定反馈 |
| 6 号电机反馈周期 | P50/P95/P99/max = 10/10/11/11 ms | 约 100 Hz 稳定反馈 |
| F103 请求响应端到端延迟 | P50/P95/P99/max = 18/24/25/25 ms | 低频请求响应稳定 |
| F103 ACK 应用层 seq 丢包 | 0 | 未观测到 seq 丢失 |
| CAN 控制器错误计数 | TEC=0，REC=0，CEL=0 | 无物理层错误累计 |
| 总线错误状态 | EW=0，EP=0，BO=0 | 无 warning、error passive、bus off |
| RX 队列溢出 | rxf0_full=0，rxf0_lost=0，rx_drain_limit=0 | 未观测到接收溢出 |
| TX 发送异常 | tx_timeout=0，tx_send_fail=0 | 未观测到发送超时或失败 |

## 4. 分节点通信质量

| CAN ID | 节点/含义 | 方向 | 帧率 | 周期 P50/P95/P99/max | 周期偏差 |
|---|---|---:|---:|---|---|
| 0x7C2 | F103 传感器数据 | RX | 50.43 fps | 20/20/20/20 ms | P95 0 ms，max 0 ms |
| 0x7C3 | F103 健康状态 | RX | 0.99 fps | 1001/1004/1006/1006 ms | P95 4 ms，max 6 ms |
| 0x321 | NanoPi 心跳 | RX | 1.02 fps | 1001/1006/1014/1014 ms | P95 6 ms，max 14 ms |
| 0x180004FD | 4 号电机主动反馈 | RX | 94.96 fps | 10/10/11/11 ms | P95 0 ms，max 1 ms |
| 0x180005FD | 5 号电机主动反馈 | RX | 94.11 fps | 10/10/11/11 ms | P95 0 ms，max 1 ms |
| 0x180006FD | 6 号电机主动反馈 | RX | 93.26 fps | 10/10/11/11 ms | P95 0 ms，max 1 ms |
| 0x330~0x334 | M33 电机状态转发 | TX | 各约 9.5 fps | N/A | N/A |

说明：3 号 CANSimple 电机未纳入本次多电机统计口径；1/2/7 号电机在本轮测试中未观测到有效反馈。

## 5. 可靠性结论

在 1 Mbps Classic CAN 条件下，系统基础传感链路和 4/5/6 号电机反馈链路均表现稳定。接入 3 路约 100 Hz 电机主动反馈后，总线利用率仅约 5.7%，未出现 CAN 控制器错误累计、bus off、RX FIFO 丢失、TX 超时或发送失败。F103 传感数据帧保持 20 ms 周期稳定传输，4/5/6 号电机反馈保持 10 ms 级周期稳定传输，说明当前 CAN 通信链路满足康复机械臂多传感、多电机状态采集的实时性和可靠性需求。

需要说明的是，电机主动反馈帧本身没有应用层 seq 字段，因此本轮电机反馈只能统计周期稳定性和总线可靠性，不能严格计算电机反馈的应用层 seq 丢包率。后续若需要进一步量化电机命令到反馈的严格端到端延迟，可在电机协议或 M33 侧增加可匹配的命令序号或时间戳。
