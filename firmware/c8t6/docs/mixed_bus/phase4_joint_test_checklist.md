# 阶段4联调检查清单（PSoC + F103 + 电机 + NanoPi/ROS）

## 1. 总线模式检查
- [ ] `can0` 运行在经典 CAN（非 FD）模式
- [ ] 波特率统一为 `1Mbps`
- [ ] 抓包中未出现 FD 样式帧（`##`）

## 2. 电机 CANopen 检查
- [ ] 5 个电机上线可见
- [ ] NMT 状态切换正常
- [ ] PDO 周期稳定，无异常风暴

## 3. F103 私有协议检查
- [ ] `0x7C0` 下发命令可收到 `0x7C1` ACK
- [ ] `0x7C2` 传感帧周期稳定（目标 100Hz）
- [ ] `0x7C3` 健康帧周期稳定（目标 1Hz）
- [ ] F103 在电机高频流量下无接收风暴（硬件过滤生效）

## 4. ROS 闭环检查
- [ ] ROS 指令 -> PSoC -> 电机 执行成功
- [ ] 电机状态与 F103 传感数据可在 ROS 同时观测
- [ ] ROS 不直接发送 F103 私有帧和电机底层细节帧

## 5. 稳定性检查（建议30分钟）
- [ ] 无节点持续离线
- [ ] 总线错误可恢复（掉线/重启后自动回到可运行态）
- [ ] F103 错误计数不持续增长（或增长可解释）

## 6. 性能指标
- [ ] 控制链路端到端延迟 < 20ms
- [ ] F103 上报周期抖动满足 100Hz 目标要求
