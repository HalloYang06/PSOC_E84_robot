# Claude NPC 消息桥接器测试指南

## 修改完成

我已经成功修改了系统，现在当你从平台发送消息给Claude NPC时，会自动弹出一个**可见的PowerShell窗口**，实时显示消息和回复！

## 修改内容

### 1. 保持了现有的Python adapter（用于Codex等其他提供方）
- Codex NPC 继续使用后台Python adapter
- 不影响现有功能

### 2. 为Claude NPC添加了可见的消息桥接器
- 当发送消息给Claude NPC时，会启动一个新的PowerShell窗口
- 窗口会实时显示：
  - 收到的平台消息
  - Claude的回复
  - 处理状态

## 测试步骤

### 1. 重启开发服务器
```bash
pnpm dev
```

### 2. 在平台发送消息
1. 访问 2D开发板页面
2. 找到你的Claude NPC
3. 点击 "打开对话框"
4. 输入消息："你好，测试消息桥接器"
5. 点击发送

### 3. 观察效果
**应该会立即弹出一个新的PowerShell窗口**，实时显示消息和回复！

## 工作原理

```
平台Web → 数据库 + 消息文件 → 消息桥接器(PowerShell窗口) → Claude CLI → 显示回复
```

## 与现有系统的区别

### 之前（Python adapter后台模式）
- ✅ 消息能发送和接收

### 现在（Claude消息桥接器）
- ✅ 消息能发送和接收
- ✅ **实时看到消息和回复**
