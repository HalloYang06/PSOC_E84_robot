灵动康复 ArmControl

康复机械臂手机端测试包。当前为内部 debug 版本，用于手机安装验证、后端登录和 App 闭环联调。

下载 APK 打开 PWA

安装提示

  1. 用安卓手机浏览器打开本页，点击“下载 APK”。
  2. 系统提示未知来源时，允许当前浏览器安装应用。
  3. 安装完成后，桌面应用名应显示为“灵动康复 ArmControl”。
