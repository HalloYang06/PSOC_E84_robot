# Unity Web 前端方案文档 - 2026-04-21

## 前端目标
前端不再承担“后台页 + 游戏皮”的职责，而是直接承担 3D 世界本体。

前端要解决 3 件事：
- 把项目管理、基地经营、NPC 协作放进同一个世界
- 让玩家能自然地看见 AI、电脑、任务、建筑和状态
- 让交互尽量发生在世界里，而不是堆在网页表单里

## 页面结构
仍然只保留 3 个页面概念，但游戏页改为 Unity Web：

1. 登录注册页
- 登录
- 注册
- 回跳参数

2. 项目管理页
- 选择项目
- 新建项目
- 接受邀请
- 查看项目摘要

3. 游戏页
- Unity Web 场景
- 世界内 UI
- 团队面板
- 电脑接入、线程扫描、AI 派发、回执查看

## Unity 场景结构
建议第一版只做一个“研发基地”主场景，不先做开放世界。

场景分区建议：
- 主控大厅：总览项目状态、目标、当前卡点
- AI 宿舍 / 办公区：放 AI-NPC
- 电脑机房：显示接入电脑与线程
- 任务大厅：显示任务池、任务流转
- 会议室：交接、需求、讨论结果
- 审批区：高风险动作审批
- 实验区：硬件实验、构建、联调

## NPC 方案
每个 NPC 对应一个 AI 成员。

每个 NPC 需要有：
- 名字
- 职位
- 外观类型
- 当前状态
- 绑定线程/电脑
- 当前任务
- 可交互入口

前端上 NPC 分成三类：
- 核心 NPC：能直接对话、接任务、给回执
- 背景 NPC：主要展示状态和位置
- 系统 NPC：比如维护员、调度员、引导员

## UI 设计原则
UI 必须轻，不再走重面板路线。

建议：
- 常驻顶部只保留最少资源与状态
- 世界交互优先用 3D 头顶标记、建筑标记、范围触发
- 深层操作通过“小面板 + 对话框 + 世界对象交互”展开
- 避免右侧大抽屉长期压住视野

## 前端交互模型
交互分 4 层：

1. 世界探索交互
- 移动
- 镜头
- 进入建筑
- 靠近 NPC

2. NPC 对话交互
- 查看当前状态
- 下发任务
- 追问进度
- 查看阻塞

3. 建筑交互
- 电脑机房：看电脑、线程、在线状态
- 任务大厅：看任务池与优先级
- 审批区：人工确认高风险动作

4. 管理面板交互
- 项目配置
- 邀请协作者
- 新建项目
- 全局团队面板

## Unity 与网页壳的边界
建议采用“网页负责壳，Unity 负责世界”的模式：

- 网页壳负责：
  - 路由
  - 登录态
  - 项目切换
  - 轻量外围表单

- Unity 负责：
  - 3D 世界
  - NPC
  - 建筑交互
  - 任务与状态可视化
  - 对话和世界内派发

## 数据同步建议
Unity 不自己发明业务数据结构，只消费后端标准接口。

前端需要的实时对象：
- 项目
- NPC 列表
- AI 工位
- 电脑节点
- 线程列表
- 任务流
- 审批流
- 交接流
- 回执消息

建议同步模式：
- 首屏 REST 拉取
- 高频状态用轮询
- 后续再升级为 WebSocket

## 首版可玩范围
首版不要做大世界，做一个“小基地 + 10 个以内 NPC”。

首版最小可玩闭环：
- 玩家进入基地
- 看到 3 到 5 个核心 NPC
- 看见 1 到 3 台接入电脑
- 扫描线程
- 把任务发给某个 NPC 背后的线程
- 收到回执
- NPC 状态变化

## 美术建议
前端美术不要自己乱做。

建议：
- 世界素材用现成 3D 基地/科幻/办公/实验室资源包
- 角色用现成低模或中低模包
- UI 用统一风格，不自己拼贴多套风格
- 第一阶段优先低多边形、低特效、轻阴影，保证 Web 流畅

## 性能建议
首版性能目标：
- 1080p 常规笔记本可运行
- 世界内 10 到 15 个 NPC 可见
- 同时真正活跃的 AI-NPC 控制在 3 到 6 个

前端优化重点：
- 降低模型面数
- 控制实时灯光
- 减少高频寻路
- 世界对象池化
- 远距离 NPC 降级

## 技术建议
推荐技术组合：
- Unity 2022/2023 LTS
- WebGL 发布
- Addressables 管理资源
- 前端桥接 JS SDK
- 后端 REST + 后续 WebSocket

## 前端阶段拆分
第一阶段：
- 3D 基地
- 登录后进入项目
- 3 个建筑
- 3 个核心 NPC
- 可对话

第二阶段：
- 电脑机房接入
- 扫描线程
- 指定线程派发
- 回执展示

第三阶段：
- NPC 默认协作
- 建筑成长
- 状态联动
- 后续多人联机预留

## 前端一句话原则
前端的任务不是把后端功能搬上屏幕，而是把后端协作系统变成一个可玩的 3D 研发基地世界。
