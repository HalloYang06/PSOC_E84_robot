# AI 合作平台第一版开发文档入口

本仓库当前目标：**开发第一版 AI 合作平台 MVP**。

第一版不是完整商业产品，不做复杂插件市场、不做完整多租户、不做真实硬件自动控制。第一版只需要打通一个核心闭环：

```text
创建项目
  -> 添加 AI
  -> 注册 Runner
  -> 创建任务
  -> 指派 AI
  -> 创建任务分支
  -> AI/Runner 辅助执行
  -> 查看日志和 diff
  -> 记录 token
  -> 审查
  -> 人类确认
  -> 合并或回滚
```

---

## 1. 所有 AI 先读这里

每个 AI 进入项目后，第一件事是确认角色。

必须先读：

1. `AI协作平台第一版开发入口.md`
2. `AI身份认领与统一协作规则.md`
3. `AI协作平台开发AI分工文档.md`

未确认身份的 AI 不允许修改代码、创建分支、提交需求单或执行 Runner 任务。
