## 当前重点问题池

- 4/5/6 active-report 是否应由 M33 安全启动逻辑管理。
- 3/4/5/6 方向、零点、传动比、软/硬限位和患者 ROM 标定。
- 1/2 腕部 4015 电机接线、协议、反馈和关节映射确认。
- C8T6 四路 EMG 接入 M33/M55 输入窗口。
- M55 验证模型替换为正式 EMG/疲劳/意图模型。
- MuJoCo conservative limit clamp 与真实物理限制的校准关系。

## 安全提醒

调试记录里如果出现真实电机动作，必须写清楚：

- 是否发过 `0x320`。
- 是否只是 bench-debug。
- 是否有人在设备内。
- M33 安全状态是否确认。
- 4/5/6 active-report 是否在验证后关闭。
- 7 号外部电机是否被错误纳入正式映射。
