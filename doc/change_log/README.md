# Change Log

本目录用于记录跨 ROS2/NanoPi/MuJoCo、M33、M55 三工程的项目级变更。它不是替代 Git commit，而是帮助人快速理解“今天为什么改、改了哪里、影响什么”。

## 记录规则

- 每次重要修改后，先提交对应仓库 Git，再在这里追加项目级摘要。
- 如果修改发生在 M55 Git 工程，还要同步更新该工程的 `docs/CHANGELOG.md`。
- 如果修改发生在非 Git 工作副本，例如当前 `wifi`，必须在记录里明确“未纳入 Git”，并尽快迁移或初始化版本管理。
- 不记录密码、token、私钥或个人凭据。

## 推荐格式

```text
## YYYY-MM-DD

### 摘要

### 涉及仓库

### Git 提交

### 安全影响

### 验证

### 后续事项
```
