# 教育版 2D 第一主线剧本文档：平衡小车 - 2026-04-30

## 1. 剧情目标

玩家来到“星桥工坊镇”，镇上的平衡小车套件失去了稳定核心。总导师 NPC 希望玩家修复它，并通过学习主控、传感器、电机、接线、PID 和烧录，让小车重新站起来。

主线名称：

```text
站起来的小车
```

主线完成标准：

```text
玩家完成模块逻辑任务、仿真接线任务、PID 调参任务，并通过烧录师 NPC 的最终验收。
```

## 2. 主要 NPC

### 总导师：林博士

职责：

```text
引导玩家接主线，解释平衡小车目标，把 AI 生成的任务分配给各 NPC。
```

开场对白：

```text
欢迎来到星桥工坊镇。这里的第一台训练小车还不能自己站稳。你不用一下子看懂所有代码，我们会把它拆成几个小任务。先去手册馆找墨书管理员，他会把小车套件资料读给你听。
```

### 手册管理员：墨书

职责：

```text
读取套件手册、展示 BOM、介绍主控和模块。
```

任务：

```text
阅读小车套件手册
确认本次项目包含主控、IMU、电机驱动、电机、电池、编码器和下载线
```

### 主控工程师：阿芯

职责：

```text
带玩家理解工程初始化、引脚配置、主循环。
```

小游戏：

```text
代码顺序拼图：把 setup/init/main loop/safety check 放到正确顺序。
```

### 传感器工程师：璃感

职责：

```text
带玩家理解 IMU 读取和姿态角。
```

小游戏：

```text
选择题：平衡小车最关心哪个角度？
选项: 温度 / 俯仰角 / 屏幕亮度 / 蓝牙名称
正确: 俯仰角
```

### 电机工程师：轮匠

职责：

```text
带玩家理解 PWM、电机方向、编码器测速。
```

小游戏：

```text
逻辑连线：把“车向前倒”“电机向前补偿”“PWM 增大”连接起来。
```

### 控制算法师：派德

职责：

```text
带玩家理解 PID 的 P/I/D 三项。
```

小游戏：

```text
参数卡片：把 P 放到“立刻纠正误差”，I 放到“修正长期偏差”，D 放到“抑制变化过快”。
```

### 仿真师：桥线

职责：

```text
带玩家完成虚拟接线。
```

小游戏：

```text
接线台：拖拽杜邦线连接 VCC/GND/SDA/SCL/PWM/DIR/ENC。
```

### 烧录师：火花

职责：

```text
编译、烧录、串口日志、最终上板确认。任何真实烧录动作都必须由玩家确认。
```

对白原则：

```text
我可以帮你检查编译结果，但烧录到真实设备前，你要确认电源、接线和板卡型号都正确。
```

## 3. 主线结构

### 第 0 章：进入星桥工坊镇

触发条件：

```text
用户完成 AI 模型选择和 API Key 配置。
```

流程：

1. 玩家出生在新手村广场。
2. 林博士说明主线目标。
3. 玩家领取“平衡小车修复委托”。
4. 系统生成任务日志。

奖励：

```text
新手工程师徽章
平衡小车项目卡
```

目标：

```text

```

1. 打开手册包。
2. 查看 BOM。
3. 点亮模块图鉴。

### 第 5 章：第一次交任务

目标：

```text

```

NPC：

```text
林博士
```

验收行为：

```text
AI 汇总玩家完成情况，编译一次阶段工程，生成“第一轮报告”。
```

通过后对白：

```text
小车已经有了大脑、感觉和轮子。下一步不是直接上电，我们先在仿真台把线接对。
```

奖励：

```text
工程学徒 Lv2
解锁仿真接线台
```

### 第 6 章：仿真接线台

目标：

```text
在虚拟环境正确连接小车模块。
```

NPC：

```text
桥线
```

玩法：

1. 左侧显示主控引脚。
2. 右侧显示 IMU、电机驱动、编码器、电源。
3. 玩家拖线连接。
4. 系统实时提示低风险错误。
5. 电源类错误只提示风险，不允许继续。

验收规则：

```text
GND 必须共地
VCC 电压必须匹配
SDA/SCL 不可反接
PWM/DIR 连接到对应电机驱动端口
编码器 A/B 接到支持中断或计数的端口
```

奖励：

```text
接线安全徽章
经验 +150
解锁实物接线指引
```

### 第 7 章：第二次交任务

NPC：

```text
林博士
```

验收：

```text
生成仿真接线报告，列出玩家接对的线、曾经犯错的线、实物接线注意事项。
```

通过后解锁：

```text
PID 调参塔
```

### 第 8 章：PID 调参塔

目标：

```text
让玩家理解并调试 P/I/D。
```

NPC：

```text
派德
```

模式：

```text
自己调参
AI 建议调参
AI 自动试调
```

界面要显示：

```text
Kp/Ki/Kd 滑杆
小车倾角曲线
电机输出曲线
震荡程度
恢复时间
AI 建议
```

AI 指导示例：

```text
现在小车反应太慢，可以稍微提高 Kp。注意一次只改一个参数，这样你才知道变化来自哪里。
```

奖励：

```text
控制精灵
经验 +200
```

### 第 9 章：烧录站

目标：

```text
编译通过，并在玩家确认后烧录到设备。
```

NPC：

```text
火花
```

流程：

1. AI 自动编译。
2. 展示编译成功或错误解释。
3. 玩家确认板卡型号、端口、电源、接线。
4. 烧录师执行烧录。
5. 读取串口日志。
6. 玩家观察小车表现。

人类确认清单：

```text
我确认板卡型号正确
我确认电源未反接
我确认电池电压符合说明
我确认轮子悬空或处于安全测试环境
我确认可以开始烧录
```

奖励：

```text
烧录通行证
经验 +150
```

当玩家完成主线后，林博士提示：

```text
你现在可以提出自己的小发明需求了。比如给小车加蓝牙遥控、OLED 显示、电量检测、避障模块，或者做一个全新的传感器项目。
```

AI 生成支线时必须给出：

```text
项目目标
器件清单
淘宝搜索关键词
接线方案
代码任务
仿真任务
验收标准
风险提醒
```
